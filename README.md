<h1>🎮 Tic Tac Toe Game</h1>

<p>A simple and interactive Tic Tac Toe game built with HTML, CSS, and JavaScript. Two players can play the game by taking turns between O and X.</p>

<h2>🚀 Features</h2>

<ul>
  <li>🎮 Two-player game</li>
  <li>🔄 Players take turns between O and X</li>
  <li>🏆 Checks all possible winning combinations</li>
  <li>🎉 Displays the winner</li>
  <li>🤝 Detects a draw when all boxes are filled</li>
  <li>🔁 Reset Game button</li>
  <li>🆕 New Game button</li>
  <li>🚫 Prevents players from selecting the same box twice</li>
  <li>📱 Responsive game board</li>
</ul>

<h2>🔄 How It Works</h2>

<ol>
  <li>The game starts with player O.</li>
  <li>Players click on an empty box to make a move.</li>
  <li>The turn changes between O and X.</li>
  <li>After every move, the game checks all winning combinations.</li>
  <li>If a player gets three matching symbols in a row, that player wins.</li>
  <li>If all nine boxes are filled without a winner, the game ends in a draw.</li>
  <li>Players can start a new game using the Reset Game or New Game button.</li>
</ol>
