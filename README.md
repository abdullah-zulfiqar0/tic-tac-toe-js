# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe game** built with **HTML, CSS, and JavaScript**.

Two players can play the game by taking turns. The game checks the winning combinations and displays the winner or a draw message.

## 🚀 Features

* Two-player game
* Players take turns between `O` and `X`
* Checks all possible winning combinations
* Displays the winner
* Detects a draw when all boxes are filled
* Reset Game button
* New Game button
* Prevents players from selecting the same box twice
* Responsive game board

## 🔄 How It Works

1. The game starts with player `O`.
2. Players click on an empty box to make a move.
3. The turn changes between `O` and `X`.
4. After every move, the game checks all winning combinations.
5. If a player gets three matching symbols in a row, that player wins.
6. If all nine boxes are filled without a winner, the game ends in a draw.
7. Players can start again using the **Reset Game** or **New Game** button.
